📊 Cost & Billing

                                            ✅ cur:DescribeReportDefinitions
                                            ✅ budgets:ViewBudget

                                            ✅ s3:ListBucket

🖥️ Resource Discovery

                                        

                                            ✅ ec2:Describe*
                                            ✅ rds:Describe*
                                            ✅ elasticloadbalancing:Describe*
                                        

                                    

                                    

                                        
📋 Optional

                                            ☑️ savingsplans:Describe*
                                            ☑️ cloudwatch:GetMetricStatistics

{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "CostBillingAccess",
      "Effect": "Allow",
      "Action": [
        "ce:*",
        "cur:DescribeReportDefinitions",
        "budgets:ViewBudget"
      ],
      "Resource": "*"
    },
    {

      "Resource": "*"
    },
    {
      "Sid": "ResourceDiscovery",
      "Effect": "Allow",
      "Action": [
        "ec2:Describe*",
        "rds:Describe*",
        "elasticloadbalancing:Describe*",
        "cloudwatch:GetMetricStatistics",
        "cloudwatch:ListMetrics",
        "organizations:Describe*",

Unlock resource-level costs — per S3 bucket, EC2 instance, RDS, Lambda and more.

Step 1 — Enable Cost & Usage Reports in AWS

                        

                            

                                1
                                Go to AWS Console → Billing & Cost Management → Cost & Usage Reports
                            

                            

                                2
                                Click "Create report"
                            

                            

                                3
                                If you see "Enable Cost Explorer" banner first — click it and wait ~24h, then come back
                            

                        

                        
                            
                            Open AWS Billing Console

Delivery Settings

                                

                                    
S3 bucketCreate new or use existing

Report format✅ text/csv

                                    
Compression✅ GZIP
